# Exclusao de Conta e Dados - Escalas Hub

Ultima atualizacao: 09/06/2026

Esta pagina explica como usuarios do app Escalas Hub podem solicitar a exclusao de conta e dados.

## Conta de usuario

Na versao atual, o Escalas Hub nao possui cadastro em servidor, login obrigatorio ou conta online vinculada ao aplicativo. Os dados informados pelo usuario sao armazenados localmente no proprio dispositivo.

## Dados armazenados localmente

O app pode armazenar localmente informacoes como perfil, documentos cadastrados, datas de vencimento, anotacoes de calendario, checklist, links salvos e respostas do formulario de avaliacao de fadiga.

## Como excluir dados no dispositivo

O usuario pode excluir dados diretamente no app quando a funcao estiver disponivel, como apagar anotacoes, documentos, links salvos e itens adicionados ao checklist.

Para remover todos os dados locais do app, o usuario pode:

- Desinstalar o aplicativo do dispositivo.
- Usar as configuracoes do Android para limpar armazenamento/dados do app Escalas Hub.

## Solicitacao por e-mail

Caso precise de ajuda ou queira solicitar orientacao sobre exclusao de dados, envie um e-mail para:

tecnautica@terra.com.br

Inclua no e-mail:

- Nome do app: Escalas Hub.
- Descricao da solicitacao.
- Informacao de contato para resposta.

Como os dados ficam localmente no aparelho e nao em servidor proprio do Escalas Hub, a exclusao efetiva dos dados locais depende de acao no proprio dispositivo do usuario.

## Dados que podem ser mantidos

Como o app nao possui servidor proprio nesta versao, o Escalas Hub nao mantem base central de contas de usuario. Se futuramente forem adicionados login, sincronizacao, compras, anuncios ou recursos corporativos com servidor, esta pagina sera atualizada com os procedimentos correspondentes.
