# Politica de Privacidade - Escalas Hub

Ultima atualizacao: 09/06/2026

Esta Politica de Privacidade descreve como o app Escalas Hub trata informacoes inseridas pelo usuario.

## Dados tratados pelo app

O Escalas Hub pode permitir que o usuario informe dados como nome, empresa, cidade, estado, profissao, documentos, datas de vencimento, anotacoes de calendario, checklist, links salvos e respostas do formulario de avaliacao de fadiga.

## Armazenamento local

Atualmente, os dados informados no app sao armazenados localmente no proprio dispositivo do usuario. O Escalas Hub nao possui, nesta versao, servidor proprio para coletar, sincronizar ou vender dados pessoais.

## Permissoes do dispositivo

O app pode solicitar permissoes para:

- Camera ou galeria, quando o usuario quiser anexar ou registrar documentos.
- Notificacoes, para lembretes relacionados a documentos e rotina.
- Abertura/compartilhamento de arquivos, quando o usuario exportar documentos ou PDFs.

Essas permissoes sao usadas apenas para executar funcoes escolhidas pelo usuario dentro do app.

## Compartilhamento de dados

O Escalas Hub nao vende dados pessoais. Arquivos exportados, PDFs, imagens ou backups so saem do dispositivo quando o proprio usuario escolhe compartilhar, salvar ou abrir esses arquivos usando recursos do sistema operacional.

## Links externos

O app pode permitir que o usuario abra links externos no navegador interno. Sites acessados pelo usuario podem ter suas proprias politicas de privacidade, independentes do Escalas Hub.

## Seguranca

Como os dados ficam no aparelho, a protecao tambem depende da seguranca do dispositivo, senha, biometria, backups e aplicativos instalados pelo usuario.

## Exclusao de dados

O usuario pode apagar anotacoes, documentos, links salvos e outros dados diretamente no app quando a funcao estiver disponivel. A desinstalacao do app tambem pode remover os dados locais armazenados pelo aplicativo, conforme o comportamento do sistema operacional.

## Alteracoes nesta politica

Esta politica pode ser atualizada conforme o app evoluir, especialmente se forem adicionadas sincronizacao em nuvem, contas de usuario, anuncios, pagamentos ou recursos corporativos.
